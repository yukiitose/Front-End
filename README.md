# UM Tagum College - IT15 Final Project

## Integrative Programming - React Frontend

### Technologies Used

- React 18
- Vite 5
- React Router DOM 6
- Axios
- Recharts
- React Icons

### Requirements

- Node.js >= 18
- npm

### Frontend Setup

```bash
cd final-frontend
npm install
cp .env.example .env
npm run dev
```

### Environment Variables

```
VITE_API_URL=http://127.0.0.1:8000/api
VITE_WEATHER_API_KEY=your_openweathermap_api_key
```

### Pages

- /login - Login page
- /dashboard - Main dashboard with charts
- /students - Students table with search
- /courses - Courses cards with filter
- /weather - Weather widget
- /forgot-password - Forgot password
- /reset-password - Reset password
- /create-password - Create password with access code
